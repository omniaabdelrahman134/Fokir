# Fokir - Personal Portfolio Website 💼

This is my second website built using pure HTML and CSS.

It is a personal portfolio template designed for a web developer.  
Compared to my first project, this one focuses more on layout organization, cleaner styling, and better visual structure.

<img width="1899" height="929" alt="image" src="https://github.com/user-attachments/assets/f76e120a-debb-41d9-9467-d8e3879acfc2" />


## 🚀 Project Overview

The website includes:

- Navigation bar
- Hero section
- About section
- Services section
- Portfolio gallery
- Statistics section
- Client feedback section
- Blog section (including embedded YouTube video)
- Contact form
- Footer with social media icons

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome
- Icofont Icons
- YouTube Embed (iframe)

## 🎯 What I Improved in This Project

- Better section spacing and layout structure
- More organized CSS
- Improved typography using multiple Google Fonts
- Icon integration using both Font Awesome and Icofont
- Portfolio image grid layout
- Embedding external content (YouTube video)
- Structured contact form design

## 📌 Notes

- Static website (no JavaScript functionality).
- Desktop layout only (not responsive yet).
- Built to strengthen my HTML & CSS fundamentals before moving to JavaScript frameworks.

## 💡 Future Improvements

- Make it fully responsive using media queries
- Improve accessibility
- Add animations and transitions
- Convert layout to Flexbox / CSS Grid
- Add JavaScript interactivity
